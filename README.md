# astsu.sh - A firewall configuration tool

## What is astsu?
It is a tool that helps you to configure the firewall, to protect yourself from ddos attacks, like SYN flood, UDP flood, low band attacks, etc.

## Usage
First install the tool
```
chmod +x install.sh
./install.sh
```

To configure type 
``` 
astsu --configure 
```
